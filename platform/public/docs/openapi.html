    Example Platform v1.0.0

    Scroll down for code samples, example requests and responses. Select a language for code samples from the tabs above or the mobile navigation menu.

    This is an example API for a platform than manages Organizations and Users.

    Base URLs:

    • https://example-third-part-platform.{cloudflare-workers-subdomain}.workers.dev/api

      • cloudflare-workers-subdomain - Default: replace-me

    License: Apache 2.0

    Authentication

    • HTTP Authentication, scheme: basic

    GET /org/{orgId}/users

    List all users

    Status Meaning Description Schema
